    Table Sys_Web_Hooks

    Entity

    Entity: Systems.Core.WebHooks

    Webhook templates. Activated by business rules or other means. Entity: Sys_Web_Hooks (Introduced in version 22.1.5.8)

    Summary

    Name Type Description
    Body nvarchar(max) The body of the POST request (interpolated string).
    Code nvarchar(32)
    Headers nvarchar(max) Optional headers for the request (interpolated string). Each header is specified on a separate line.
    Name nvarchar(254) ML Webhook name (multi-language).
    Notes nvarchar(max)
    Repository_Name nvarchar(128) The name of the repository for which the webhook is setup. All interpolated strings will be evaluated in the context of entities of the specified type.
    Retry_Logic nvarchar(1) Allowed: 0, 3 0=Don't retry (default); 3=Retry up to 3 times
    Row_Version timestamp
    URL nvarchar(2000) The destination URL, which should be called by the webhook (interpolated string).
    Web_Hook_Id uniqueidentifier PK
