    Line_Deal_Type_Id uniqueidentifier Deal type to be passed to the invoice line. If deal type in entered then the invoice creates VAT entry for this deal type.

    Line_No int Consecutive number of the line within the sales order
    Line_Standard_Discount_Percent decimal(7, 6) Readonly Standard discount for the line. This is automatically computed according to discount conditions
    Line_Store_Id uniqueidentifier The store which should be used to issue the goods for the line. NULL means to use the store from the header
    Line_To_Date date When selling a service valid only for a period, denotes the end of the period. NULL means that it is unknown or N/A.
    Lot_Id uniqueidentifier Specifies the lot from which the goods should be issued. NULL means that the lot will be specified at a later stage (store order, etc.)
    Notes nvarchar(max)
    Parent_Document_Id uniqueidentifier The document, which the current line executes. NULL when the current line does not execute another line.
    Parent_Line_No int The number of the line within the parent document, which the current line executes. NULL when the current line does not execute parent line.
    Persist_Lot bit If checked specifies that the lot in the line cannot be changed in the sub-documents created by the current document.
    Product_Code_Id uniqueidentifier Used to set the Product_Id thru the coding systems
    Product_Description nvarchar(254) ML The name of the sold product at the time the sale was made
    Product_Id uniqueidentifier The product sold
    Product_Price_Id uniqueidentifier Not NULL when the price has been selected from the list of valid standard prices
    Product_Variant_Id uniqueidentifier If specified determines which product variant of the current product in this line is used.
    Promotional_Package_Id uniqueidentifier Readonly The promotional package, based on which the line was added. NULL when the line was not added as part of a promotional package
    Quantity decimal(12, 3) The quantity sold
    Quantity_Base decimal(12, 3) The equivalent of Quantity in the base measurement category of the product
    Quantity_Unit_Id uniqueidentifier The measurement unit of Quantity
    Requested_Quantity decimal(12, 3) Quantity requested by customer
    Required_Delivery_Date date The required (contracted) delivery date for the line
    Return_For_Invoice_Line_Id uniqueidentifier When specified, indicates that the current line is a return for products, invoiced with the specified invoice line.
    Return_For_Sales_Order_Line_Id uniqueidentifier When specified indicates that the goods sold in Return_For_Sales_Order_Line_Id are returned with the current line
    Row_Version timestamp
    Sales_Order_Id uniqueidentifier
    Sales_Order_Line_Id uniqueidentifier PK
    Serial_Number_Id uniqueidentifier Which serial number to receive/issue. NULL means that serial number is unknown or not applicable
    Standard_Quantity_Base decimal(12, 3) Readonly The theoretical quantity in base measurement unit according to the current measurement dimensions for the product. Used to measure the execution.
    Standard_Unit_Price decimal(14, 5) Readonly Standard unit price of the product during the creation of the sales order line
    Store_Bin_Id uniqueidentifier The bin from which the goods should be withdrawn. NULL means that the bin will be specified at a later stage (store order, etc.)
    Unit_Price decimal(14, 5) Unit price of the product in the currency of the sales order and in the unit of measure, as specified by QuantityUnitId
