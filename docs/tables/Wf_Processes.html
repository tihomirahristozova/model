    Table Wf_Processes

    Entity

    Entity: Systems.Workflow.Processes

    Contains the business process diagrams. Entity: Wf_Processes

    Summary

    Name Type Description
    Creation_Time datetime Readonly
    Creation_User nvarchar(64) Readonly
    Is_Landscape bit Specifies whether the process diagram is intended to be viewed in landscape mode.
    Notes nvarchar(2000)
    Process_Id uniqueidentifier PK
    Process_Name nvarchar(128) ML
    Row_Version timestamp
    Schema_Format nvarchar(1) Application specific format of the Schema Layout.
    Schema_Layout nvarchar(max) Contains the actual presentation layout of the business process. The layout is stored in the format, specified by Schema Format.
    Start_Event nvarchar(3) USR=User created; EML=Email receive (still not supported). NULL means that there is no starting event for this process
    Start_Role_Id uniqueidentifier When Start_Event='USR' then specifies the role which the user must play in order to start the process. NULL when Start_Event<>'USR'
    Thumbnail varbinary Contains the visual thumbnail of the presentation of the business process. It is stored in bitmap (BMP) format.
    Update_Time datetime Readonly
    Update_User nvarchar(64) Readonly
