    Note

    OBSOLETE! Do not use!

    Entity

    Entity: Applications.Cms.NewsWebModuleArticles

    Represents an article in a news web module. Entity: Cms_News_Web_Module_Articles (Obsoleted in version 22.1.6.73)

    Summary

    Name Type Description
    Article_Text_Html nvarchar(max) The actual article text, stored as Html. Only the of the Html is stored. The language of the text is specified in Language Code.
    Author_Public_User_Id uniqueidentifier The author of the news article.
    Is_Published bit Specifies whether the article is published for display on web.
    Language_Code nvarchar(8) The language code of the article text.
    News_Web_Module_Article_Id uniqueidentifier PK
    News_Web_Module_Id uniqueidentifier The News Web Module, to which the current article is added.
    Publish_Date datetime The date and time when the news article should appear to be published.
