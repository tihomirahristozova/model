    Table Bpm_Process_Nodes

    Entity

    Entity: Systems.Bpm.ProcessNodes

    Contains the flow Nodes of the process model. Entity: Bpm_Process_Nodes

    Owner Tables Hierarchy

    Summary

    Name Type Description
    Instructions_Html nvarchar(max) Detailed instructions to the executor in HTML format.
    Node_Type nvarchar(1) Allowed: A, E, G, C Type of the node. A=Text Annotation; E=Event; G=Gateway; T=Task; P=Sub-Process.
