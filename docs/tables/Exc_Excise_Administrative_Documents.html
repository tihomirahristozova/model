    Table Exc_Excise_Administrative_Documents

    Entity

    Entity: Finance.Excise.ExciseAdministrativeDocuments

    Represents the data, sent to the customs authorities for the movements of excise goods. Entity: Exc_Excise_Administrative_Documents (Introduced in version 21.1.3.54)

    Owner Tables Hierarchy

    Accrue_Excise_Duty bit Accrue Excise Duty
    Administrative_Reference_Code nvarchar(16) The reference code, returned the customs authorities, when the document is exported to them. NULL means the document is not yet exported to the authorities.
    Direction nvarchar(1) Allowed: I, R Indicates whether the document is Receipt('R') or Issue('I') of the Tax Warehouse
    Document_Id uniqueidentifier
