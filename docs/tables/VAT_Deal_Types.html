    Table VAT_Deal_Types

    Entity

    Entity: Finance.Vat.DealTypes

    Contains deal types supported by the system as well as user-defined types. Entity: VAT_Deal_Types

    Summary

    Name Type Description
    Country_Id uniqueidentifier The country for which this deal type applies. The country is matched against the country of the enterprise company.
    Deal_Type_Code nvarchar(32) Code of the deal type.
    Deal_Type_Id uniqueidentifier PK Unique Id for the deal type. Deal_Type_Id should be used for identifyind and updating system deal types.
    Deal_Type_Name nvarchar(254) Description of the deal type.
    Entry_Type nvarchar(1) Allowed: P, S Type of the VAT entries, which are made for this deal type. S=Sales, P=Purchases.
    Is_System bit Readonly Is_System is True for those deal types that are managed by the system via update procedures and cannot be edited by the user.
    Row_Version timestamp
    Tax_Code nvarchar(3) Allowed: STD, RED, SPR, INT, EXM, NS VAT rate type for this deal type. Can be among "STD"(Standard rate), "RED"(Reduced rate), "SPR"(Super-reduced rates), "INT"(Intermediary (Parking) rates), "EXM"(Tax Exempt), "NS"(Non-subject to tax).
