    Table Ast_Assets

    Entity

    Entity: Finance.Assets.Assets

    The fixed assets. Entity: Ast_Assets

    Summary

    Name Type Description
    Asset_Category_Id uniqueidentifier The category of this asset. The category may determine default values for the depreciation plans for the different valuation models for this asset.
    Asset_Code nvarchar(20)
    Asset_Group_Id uniqueidentifier The asset group to which this asset belongs.
    Asset_Id uniqueidentifier PK
    Asset_Name nvarchar(254) ML
    Deployment_Date datetime Date, when the asset is first deployed. Null if the asset is not deployed yet
    Enterprise_Company_Id uniqueidentifier
    Is_Active bit Checked if the assets is active and can be used in documents. The inactive assets can be used only for reports.
    Notes nvarchar(254)
    Primary_Valuation_Model_Id uniqueidentifier Obsolete. Not used.
    Purchase_Date datetime Original purchase date of the asset.
    Row_Version timestamp
    Serial_Number nvarchar(30) Serial number of the asset.
    Valuation_Currency_Id uniqueidentifier Currency in which changes of the asset values (purchase value, depreciation value and etc.) are stored.
