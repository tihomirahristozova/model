    View Dmv_Exec_Stats

    Entity

    Entity: Systems.Dmv.ExecStats

    Execution statistics dynamic management view. Entity: Dmv_Exec_Stats (Introduced in version 23.1.0.19)

    Summary

    Name Type Description
    Application nvarchar(64) The name of the application that executes the operation
    Avg_Time_Ms float Average time of operation execution
    Count int The number of times the operation is executed since last statistics reset.
    Database nvarchar(64) The database in which the operation is executed
    Is_Long_Polling bit True if the operation is long polling.
    Kind nvarchar(64) The operation kind. Various operation kinds exist, e.g. Db, Long Procs etc.
    Max_Time_Ms float The maximum time of one operation execution
    Operation nvarchar(128) The operation name
    Statistics_Since datetime The date and time since when the statistics are collected
    Total_Time_Ms float Total time spent for the operation
