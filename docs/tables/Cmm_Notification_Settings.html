    Table Cmm_Notification_Settings

    Entity

    Entity: Communities.NotificationSettings

    User notification settings for the different notification classes. Entity: Cmm_Notification_Settings (Introduced in version 20.1)

    Summary

    Name Type Description
    Notification_Class nvarchar(64) The class of the notification, for which the user is specifying settings. When NULL, the setting is applied to all notification classes.
    Notification_Setting_Id uniqueidentifier PK
    Receive_Mail bit True if the user should receive mail.
    Receive_Notification bit True if the user should receive internal notification.
    Receive_Sms bit True if the user should receive SMS.
    Row_Version timestamp
    User_Id uniqueidentifier The user for whom the notification setting is about.
