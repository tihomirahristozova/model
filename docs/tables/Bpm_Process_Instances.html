    Table Bpm_Process_Instances

    Entity

    Entity: Systems.Bpm.ProcessInstances

    Contains the running instances of the processes. Entity: Bpm_Process_Instances

    Summary

    Name Type Description
    Creation_Time datetime Readonly The date and time, when the process instance was created.
    Finish_Time datetime Readonly The date and time, when the process instance has finished its execution. NULL when the instance has not yet finished execution.
    Process_Id uniqueidentifier The process on which this instance is based.
    Process_Instance_Id uniqueidentifier PK
    Row_Version timestamp
    Start_Time datetime Readonly The date and time, when the process instance was started.
    State int Allowed: 0, 10, 20, 60, Readonly Instance state. 0=Created; 60=Completed.
    Subject nvarchar(512) The multilanguage subject (or title) of this process instance.
