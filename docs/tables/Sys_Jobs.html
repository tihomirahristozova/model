    Table Sys_Jobs

    Entity

    Entity: Systems.Core.Jobs

    The jobs, which are configured in the system. Entity: Sys_Jobs (Introduced in version 20.1)

    Summary

    Name Type Description
    Job_Id uniqueidentifier PK
    Job_Type nvarchar(3) Allowed: DOC, POS, DNT, DPI The system type of the job. DOC=Document Change State, POS=Run Postponed Events, DNT=Delete Old Notifications.
    Job_Name nvarchar(254) The name of the job.
    Is_Active bit Specifies whether the job is active and ready for running.
    Notes nvarchar(max)
    Run_On_Idle bit Specifies whether to automatically run the job when the server is idle.
    Row_Version timestamp
