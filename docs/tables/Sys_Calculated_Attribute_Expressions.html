    Table Sys_Calculated_Attribute_Expressions

    Represent an expression within the calculation of a calculated attribute. Entity: Sys_Calculated_Attribute_Expressions

    Owner Tables Hierarchy

    Expression_No int Unique number of the expression within the attribute. Expression No 0 is the default expression, which provides the result value of the attribute.
