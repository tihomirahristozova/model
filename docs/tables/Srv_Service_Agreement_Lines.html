    Table Srv_Service_Agreement_Lines

    Entity

    Entity: Applications.Service.ServiceAgreementLines

    Service agreement lines represent the individual serviced objects that are covered by the service agreement. Entity: Srv_Service_Agreement_Lines

    Owner Tables Hierarchy

    • Srv_Service_Agreements
    • Gen_Documents

    Summary

    Name Type Description
    Line_End_Date_Time datetime The ending date and time of the agreement coverage for the current line.
    Line_No int Consecutive line number, unique within the document. Usually is increasing in steps of 10, like in 10, 20, 30, etc.
    Line_Start_Date_Time datetime The starting date and time of the agreement coverage for the current line.
    Notes nvarchar(254)
    Parent_Document_Id uniqueidentifier The document, which the current line executes. NULL when the current line does not execute another line.
    Parent_Line_No int The number of the line within the parent document, which the current line executes. NULL when the current line does not execute another line.
    Quantity decimal(14, 3) The quantity of the service object that is included in the agreement
    Row_Version timestamp
    Service_Agreement_Id uniqueidentifier
    Service_Agreement_Object_Id uniqueidentifier PK
    Service_Object_Id uniqueidentifier The service object, which is covered by the current agreement.
    Service_Type_Id uniqueidentifier The type (level) of service that is agreed
