    Table Log_Shipment_Order_Lines

    Entity

    Entity: Logistics.Shipment.ShipmentOrderLines

    Represents detail lines of orders for shipment of sales orders to customers. Each line orders the shipment of one product. Entity: Log_Shipment_Order_Lines

    Owner Tables Hierarchy

    • Log_Shipment_Orders
    • Gen_Documents

    Summary

    Name Type Description
    __Object_Version int
    Box_Count int The count of boxes in which the goods are packaged. NULL means unknown
    Finished bit 1 if this shipment should prohibit further shipments for the sales order line
    Gross_Weight_kg decimal(12, 3) The gross weight of the whole line in kilogramms. NULL means unknown
    Guarantee_Period_Days int Guarantee period in days for the offered product. NULL for non-serviced products
    Height_m decimal(12, 3) The height of the package in meters. NULL means unknown
    Length_m decimal(12, 3) The length of the package in meters. NULL means unknown
    Line_No int Line number, unique for the shipment
    Lot_Id uniqueidentifier The store lot from which to take the goods
    Net_Weight_kg decimal(12, 3) The net weight of the entire batch of goods in kilograms. NULL means unknown
    Notes nvarchar(max)
    Pallet_No int The number of the pallet in which the goods are packaged. NULL means unknown
    Parent_Document_Id uniqueidentifier The document, which the current line executes. NULL when the current line does not execute another line.
    Parent_Line_No int The number of the line within the parent document, which the current line executes. NULL when the current line does not execute parent line.
    Parent_Sales_Order_Line_Id uniqueidentifier Sales order line which is shipped
    Persist_Lot bit If checked specifies that the lot in the line cannot be changed in the sub-documents created by the current document.
    Quantity decimal(12, 3) The quantity to be shipped
    Quantity_Base decimal(12, 3) The equivalence of Quantity in the base measurement category of the product.
    Quantity_Unit_Id uniqueidentifier The measurement unit of Quantity
    Row_Version timestamp

    Standard_Quantity_Base decimal(12, 3) Readonly The theoretical quantity in base measurement unit according to the current measurement dimensions for the product. Used to measure the execution.

    Volume_l decimal(12, 3) The volume in litres of the whole batch. NULL means unknown
    Width_m decimal(12, 3) The width of the package in meters. NULL means unknown
