    Table Gen_Document_Type_Amounts

    Entity

    Entity: General.DocumentTypeAmounts

    Specifies amount types, that should be automatically added to documents of a given type. Entity: Gen_Document_Type_Amounts

    Owner Tables Hierarchy

    • Gen_Document_Types

    Summary

    Name Type Description
    Default_Percent decimal(7, 6) Default input percent. Valid only for amount types, supporting percent and takes precedence over Default_Percent in the definition of the amount type
    Document_Amount_Type_Id uniqueidentifier The amount type that should be automatically added to the documents of the specified type
    Document_Type_Amount_Id uniqueidentifier PK
    Document_Type_Id uniqueidentifier The document type for which the amount type is specified
    Required_From_Date date When not null, specifies a date, after which the amount becomes required for the current document type. The date is compared against the document date.
    Required_Thru_Date date When not null, specifies a date, up to which the amount is required for the current document type. The date is compared against the document date.
    Row_Version timestamp
    User_Can_Change_Input bit 1 if the user, entering the document is allowed to change the default input percent
