    Table Cmm_Social_Reactions

    Entity

    Entity: Communities.Social.Reactions

    Reaction of a user to any social comment to an object of the system. Entity: Cmm_Social_Reactions (Introduced in version 20.1)

    Summary

    Name Type Description
    Creation_Time_Utc datetime The exact server time (in UTC), when the reaction was created.
    Data_Object_Id uniqueidentifier The root data object (post, marketplace product, document, etc), for which the reaction is.
    Reaction_Type nvarchar(3) Allowed: LIK, LOV, HAH, WOW, SAD, ANG The type of the reaction. LIK = Like; LOV = Love; HAH = Haha; WOW = Wow; SAD = Sad; ANG = Angry.
    Row_Version timestamp
    Social_Comment_Id uniqueidentifier When not null, specifies that the reaction is for the specified comment within the data object.
    Social_Reaction_Id uniqueidentifier PK
    User_Id uniqueidentifier The user, who reacted.
