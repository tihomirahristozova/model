    Begin_Date_Time datetime Not used.

    Serial_Number_Id uniqueidentifier If not NULL, specifies that the product was (has to be) stored with specific serial number

    Work_Order_Item_Operation_Id uniqueidentifier The operation for which this output is recorded.
    Workgroup_Resource_Id uniqueidentifier The resource, which was used to perform the operation.
