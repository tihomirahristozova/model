    Table Sys_Web_Sites

    Entity

    Entity: Systems.Core.WebSites

    Contains the web sites, which are hosted for the database. Entity: Sys_Web_Sites (Introduced in version 19.1)

    Summary

    Name Type Description
    Enterprise_Company_Id uniqueidentifier The enterprise company, for which is the site. NULL means, that the web site should not be enterprise company specific.
    Is_Active bit Indicates whether the web site is active and will be instantiated upon next web server restart.
    Is_Private bit Specifies that the web site address will not be publicly listed. The web site itself will still be publicly accessible; only its URL would not be listed in the auto-discovery service.
    Notes nvarchar(max)
    Relative_Url nvarchar(254) The relative Url of the site. This is the text after the first slash after the protocol and host name. The text should not include the protocol and host name. NULL means that the site will be hosted as the root site in the speicified web host.
    Row_Version timestamp
    Settings_Json nvarchar(max) The field specifies the JSON settings for this website. NULL means that there are no specific settings for this website.
    Trusted_Application_Id uniqueidentifier The trusted application related to this web site
    Web_Host_Id uniqueidentifier The web host in which to host the site.
    Web_Site_Id uniqueidentifier PK
    Web_Site_Type nvarchar(3) Allowed: API, CC, EC, LEG, SI, DM, APP, TAP, DAP The type of web site - Api, Client Center, Id, etc.
