R33172 WarehousePolicies - ZoneIsFromTheSameWarehouse

    Name Value
    Code R33172
    Entity Logistics.Wms.WarehousePolicies Entity
    Name Zone Is From The Same Warehouse
    Attribute Value
    Layer Back-End
    Events ClientCommit
    Priority Normal
    Modify NO
    Applicable Legislations ALL // no condition needed
    Action Validate that:
    If ((Warehouse != null) AND (Zone != null)), then WarehousePolicy.Zone.Warehouse = WarehousePolicy.Warehouse
    Description The rule validates that if there is a Managed Warehouse and Zone selected in the Warehouse Policy, then the Managed Warehouse of the Zone corresponds to the Managed Warehouse in the policy.
    Message Warehouse policy '{WarehousePolicy}' contains an invalid Zone.
    The selected zone '{Zone}' is not in the same Warehouse as the one specified in the policy, which is not allowed.
    Select a zone from the same warehouse.
