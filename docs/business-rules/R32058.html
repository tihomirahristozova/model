R32058 ObjectFile - Embedded File Contents Max Size

    Code R32058
    Entity Systems.Core.ObjectFiles Entity
    Name EmbeddedFileContentsMaxSize
    Attribute EmbeddedFileContents
    Layer Back-End
    Events Commit
    Modify NO
    Action Validate that: If the field ObjectFiles.EmbeddedFileContents has value than it must contain  =< 50MB data
    Description If in a panel Object:Files the field Embedded File Contents has value then it must contain up to 50MB of data. The field Embedded File Contents contains the contents of the file when it is embedded in the database. The field is null for linked files and Internet Urls.
    Message "The size of the file exceeds maximum allowed size (50 MB). Please reduce the size of the file in the limit using compression or conversion appropriately."
    Introduced In Version
    Revocable NO
